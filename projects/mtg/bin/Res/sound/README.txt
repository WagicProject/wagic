Track0.mp3, Track1.mp3, and Track2.mp3 have to be in this folder for the music options to be available.
These file must not contain any ID3v2 tag, or they won't play on the PSP.

You may add up to 20 deck music files in here as well. the files will need to be titled deck_music(#).mp3

Example

deck_music1.mp3
deck_music2.mp3

and so on.
